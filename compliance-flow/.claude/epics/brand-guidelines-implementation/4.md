# Task: Create Nomu logo and theme toggle components

## Description
Create two new React components: a Nomu logo SVG component (inline SVG using currentColor for theme adaptation) and a theme toggle button (sun/moon icon from lucide-react that calls the Zustand theme store).

## Acceptance Criteria
- [ ] `NomuLogo.tsx` renders the NOMU wordmark + circular logomark as inline SVG
- [ ] Logo uses `currentColor` so it automatically adapts to dark/light theme
- [ ] Logo accepts `className` prop for sizing flexibility
- [ ] `ThemeToggle.tsx` renders Sun icon (in dark mode) / Moon icon (in light mode) from lucide-react
- [ ] Toggle button calls `toggleTheme()` from the theme store
- [ ] Toggle has proper aria-label ("Switch to light mode" / "Switch to dark mode")
- [ ] Both components are typed with TypeScript

## Technical Details
- **New file**: `src/components/common/NomuLogo.tsx`
  - Inline SVG based on brand guide: "NOMU" text in bold geometric sans + circular logomark (mountain/wave icon inside circle)
  - Props: `className?: string`, `showWordmark?: boolean` (for compact mode)
- **New file**: `src/components/common/ThemeToggle.tsx`
  - Uses `Sun` and `Moon` icons from lucide-react
  - Reads theme from `useThemeStore()`
  - Styled as ghost button with brand hover color
