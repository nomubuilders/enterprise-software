# Task: Rebrand node components

## Description
Update all 6 node components to use brand-harmonized colors while maintaining visual distinction between node types. Apply theme-aware styling to BaseNode and update each specific node type with its brand color assignment.

## Acceptance Criteria
- [ ] **BaseNode.tsx**: Border, shadow, and selected-state glow use brand colors. Background adapts to theme. Text colors use brand tokens
- [ ] **TriggerNode.tsx**: All trigger types (Manual, Schedule, Webhook) use brand orange `#FF6C1D` as their accent color
- [ ] **DatabaseNode.tsx**: PostgreSQL, MySQL, MongoDB nodes use brand purple `#4004DA` as accent. Individual DB types remain distinguishable via icon/label
- [ ] **LLMNode.tsx**: AI Agent node uses brand purple with a subtle gradient effect for premium feel
- [ ] **PIIFilterNode.tsx**: Uses dark gray `#36312E` background with orange `#FF6C1D` border for compliance emphasis
- [ ] **OutputNode.tsx**: Chat uses purple, Email uses orange, Spreadsheet and Telegram use derived brand shades
- [ ] All nodes remain visually distinguishable from each other at a glance
- [ ] Nodes render correctly in both dark and light themes
- [ ] No remaining `slate-`, `green-`, `blue-`, `amber-`, `cyan-`, `emerald-`, `rose-`, `sky-` generic Tailwind color classes
