# Task: Visual QA and accessibility audit

## Description
Comprehensive visual review and accessibility audit across both dark and light themes. Verify all components render correctly, no leftover generic colors remain, contrast ratios pass WCAG AA, and the overall brand experience is cohesive.

## Acceptance Criteria
- [ ] Zero remaining `slate-` color classes across all source files (grep verification)
- [ ] Zero remaining generic `purple-` color classes (should all be brand-specific)
- [ ] All text/background combinations meet WCAG 2.1 AA contrast ratio (4.5:1 for normal text, 3:1 for large text)
- [ ] Focus indicators visible and branded in both themes
- [ ] Theme toggle works end-to-end: click toggle → all surfaces/text/borders update instantly
- [ ] Theme persists across page refresh
- [ ] First visit respects system `prefers-color-scheme`
- [ ] Fonts render correctly (Barlow for headings, Work Sans for body) with no visible layout shift
- [ ] All node types distinguishable from each other in both themes
- [ ] All modals, panels, sidebar render correctly in both themes
- [ ] PWA splash screen and install experience reflect brand
- [ ] No visual regressions in workflow building functionality

## Technical Details
### Verification Steps
1. **Color audit**: Run `grep -r "slate-\|purple-600\|purple-500\|purple-400" src/` to find leftover classes
2. **Contrast check**: Use browser DevTools or axe-core to verify contrast ratios
3. **Theme toggle test**: Toggle rapidly, check for flash of unstyled content
4. **Persistence test**: Set theme, close tab, reopen — theme should persist
5. **System pref test**: Clear localStorage, set OS to light mode, reload — should show light theme
6. **Font test**: Disable cache, reload — fonts should load with swap (no blank text)
7. **Node test**: Create a workflow with all node types, verify visual distinction
8. **Responsive check**: Resize browser to various widths, verify no layout breaks

### Tools
- Chrome DevTools Lighthouse (accessibility score >= 90)
- axe-core browser extension for detailed contrast issues
- Manual visual comparison of all component states
