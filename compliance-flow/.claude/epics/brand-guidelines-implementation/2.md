# Task: Set up theme infrastructure

## Description
Create the foundational theming system: CSS custom properties for all Nomu brand colors, a Zustand theme store with dark/light mode + localStorage persistence + system preference detection, Google Fonts imports in index.html, and font-family declarations in index.css.

## Acceptance Criteria
- [ ] CSS custom properties defined in `src/index.css` for all brand colors (primary #4004DA, accent #FF6C1D, backgrounds, text, surfaces)
- [ ] Light/dark theme variants toggle via class on `<html>` element
- [ ] Zustand store created at `src/store/themeStore.ts` with: currentTheme, toggleTheme, setTheme
- [ ] Theme persists to localStorage under key `nomu-theme`
- [ ] First visit respects `prefers-color-scheme` media query
- [ ] Google Fonts link tags added to `index.html` for Barlow (400,500,700) and Work Sans (400,700)
- [ ] `font-display: swap` used to prevent render blocking
- [ ] Font-family declarations in index.css: Barlow for headings, Work Sans for body
- [ ] Scrollbar colors in index.css updated from slate to brand colors (theme-aware)
- [ ] React Flow font-family override updated to use Work Sans

## Technical Details
- **New file**: `src/store/themeStore.ts` — Zustand store
- **Modified**: `index.html` — Add `<link>` for Google Fonts, update theme-color meta, change body class from `bg-slate-950` to theme-aware
- **Modified**: `src/index.css` — Add CSS custom properties block, update font declarations, update scrollbar colors, update React Flow overrides

### CSS Custom Properties Structure
```css
:root {
  --nomu-primary: #4004DA;
  --nomu-accent: #FF6C1D;
  --nomu-bg: #000000;
  --nomu-surface: #36312E;
  --nomu-text: #FEFCFD;
  --nomu-text-muted: #4D4D4D;
  --nomu-border: #4D4D4D;
}

:root.light {
  --nomu-bg: #FEFCFD;
  --nomu-surface: #FFFFFF;
  --nomu-text: #000000;
  --nomu-text-muted: #4D4D4D;
  --nomu-border: #E5E5E5;
}
```
